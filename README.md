# Full-Stack Person Management System

## Project Description
This project is a full-stack web application developed with React, Node.js (Express), PostgreSQL, and Docker Compose.

The application allows users to:
- Add new people
- View all people
- Update existing people
- Delete people

## Technologies
- React
- Node.js
- Express
- PostgreSQL
- Docker
- Docker Compose

## Setup and Run Instructions
docker compose up --build

Frontend

http://localhost:5173

Backend

http://localhost:5000

API Endpoints
GET /api/people
GET /api/people/:id
POST /api/people
PUT /api/people/:id
DELETE /api/people/:id
Validation Rules
Full Name is required
Email is required
Email must be valid
Email must be unique
